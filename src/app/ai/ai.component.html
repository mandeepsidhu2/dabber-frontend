Usage of Face-AI

Face-AI will help the user to enter into the parking system, without scanning their RF card. We will do so by mapping the RF id number against each user, and by using either of them, i.e RF card or face recognition, the user can be authorised and therefore charged.In case of the latter, the balance will be automatically charged on the card.
This is a working prototype of the face recognition of a user and how it may be used to implement the features described above.By having a photograph of the user previously with ourselves, we can recognise the user face with cameras installed in the parking system, more the number of reference images we have of the user, higher is the probability of the AI to correctly detect the user face.
In case the user has apprehensions regarding us having their photograph through the suvidham platform or otherwise, they may continue to use the RF id card.

How does it work ?

For each of the persons listed, we have their two images.More the number of images of an individual , better are the chances to correctly recognise their face. The number next to the face of the person represents how sure the AI is about correctly recognising it, where 1 represents that it is completely sure.
